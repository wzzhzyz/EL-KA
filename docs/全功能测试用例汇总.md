## 三、`/link` 接口测试（12 条）

### 3.1 正常场景

| 编号 | 场景 | text | options | 预期 |
|------|------|------|---------|------|
| **TC_LINK_001** | 基准：单实体文本 | `"国家电网发布年度报告"` | `{}` | 200, NER 识别→链接, 返回 `trace_id` + `results` + `stats`, `results` 含 `mention/type/entity_id/standard_entity/confidence/is_nil/evidence` |
| **TC_LINK_002** | 多实体文本 | `"国家电网与南方电网签署合作协议"` | `{}` | 200, 每个实体独立链接, `stats.total_mentions ≥ 2` |
| **TC_LINK_003** | 无实体文本 | `"今天天气很好"` | `{}` | 200, `stats.total_mentions = 0`, `results = []` |
| **TC_LINK_004** | 含代词文本 | `"华为发布新手机，该公司表示..."` | `{}` | 200, NER 输出含代词或不含（取决于后端） |

### 3.2 参数组合

| 编号 | 场景 | nil_threshold | enable_coreference | enable_llm_fallback | linkable_types | 预期 |
|------|------|:---:|:---:|:---:|------|------|
| **TC_LINK_005** | 默认阈值 | 0.8 (默认) | false | false | 全部 | 标准链接行为 |
| **TC_LINK_006** | 最低阈值 | 0.0 | false | false | 全部 | 所有候选均链接, NIL≈0 |
| **TC_LINK_007** | 最高阈值 | 1.0 | false | false | 全部 | 仅精确别名命中可链接, 其余 NIL |

## 四、`/link_with_mentions` 接口测试（20 条）

### 4.1 正常场景

| 编号 | 场景 | mentions 特征 | 预期 |
|------|------|------|------|
| **TC_LWM_001** | 单 mention 精确匹配 | `[{"mention":"国家电网","type":"ORG","char_start":0,"char_end":4,"confidence":1.0}]` | 精确别名命中, `is_nil=false`, `confidence≥0.9` |
| **TC_LWM_002** | 单 mention 模糊匹配 | `[{"mention":"国网","type":"ORG","char_start":0,"char_end":2,"confidence":0.9}]` | 模糊匹配成功, `evidence` 含 `alias_fuzzy` |
| **TC_LWM_003** | 单 mention NIL | `[{"mention":"完全不存在的实体XYZ","type":"ORG","char_start":0,"char_end":8,"confidence":0.5}]` | `is_nil=true`, `entity_id=""`, `standard_entity=""` |
| **TC_LWM_004** | 多 mention 混合 | 3 个 mention（2 KB内 + 1 KB外） | `stats.linked=2`, `stats.nil=1` |
| **TC_LWM_005** | 空 mention 列表 | `[]` | 200, `stats.total_mentions=0` |
| **TC_LWM_006** | 字符串格式 mentions | `"国家电网,华为,特斯拉"` | 200, 按逗号拆分, type=UNKNOWN, 各自链接 |

### 4.2 NIL 阈值参数组合

| 编号 | nil_threshold | 场景 | 预期关键行为 |
|------|:---:|------|------|
| **TC_LWM_007** | 0.65 | 低阈值 | 模糊候选可通过, NIL 判定宽松, 可能误链接 KB 外实体 |
| **TC_LWM_008** | 0.80 | 默认阈值 | 标准 NIL 判定 |
| **TC_LWM_009** | 0.90 | 高阈值(推荐) | 严格 NIL 判定, 低置信模糊候选被拦截, 精确别名不受影响 |
| **TC_LWM_010** | 0.95 | 极高阈值 | 仅精确别名命中可链接, 大部分模糊匹配→NIL |
| **TC_LWM_011** | 0.0 | 边界-零值 | 所有候选均通过, NIL≈0（极端宽松） |
| **TC_LWM_012** | 1.0 | 边界-最大值 | 仅 perfect match 通过（极端严格） |

### 4.3 功能开关组合

| 编号 | nil_threshold | coref | llm | 场景 |
|------|:---:|:---:|:---:|------|
| **TC_LWM_013** | 0.8 | false | false | 基准：纯规则链路 |
| **TC_LWM_014** | 0.8 | true | false | 规则链路 + 共指消解 |
| **TC_LWM_015** | 0.8 | false | true | 规则链路 + LLM 兜底 |
| **TC_LWM_016** | 0.8 | true | true | 全功能开启 |

### 4.4 类型过滤

| 编号 | linkable_types | 输入 mention 类型 | 预期 |
|------|------|------|------|
| **TC_LWM_017** | `["ORG"]` | ORG + PERSON | 仅 ORG 被链接, PERSON 被跳过或标记 |
| **TC_LWM_018** | `["ORG","GPE"]` | ORG + PERSON + GPE | ORG 和 GPE 被链接, PERSON 跳过 |
| **TC_LWM_019** | 不传 / `[]` | ORG + PERSON | 全部类型参与链接 |
| **TC_LWM_020** | `["UNKNOWN_TYPE"]` | ORG | 无匹配类型, 全部跳过或全部 NIL |

## 十、参数组合矩阵总览

```
                    nil_threshold
                    ├── 0.0  (TC_LWM_011)
                    ├── 0.65 (TC_LWM_007)
                    ├── 0.80 (TC_LWM_001~006, TC_LWM_013)
                    ├── 0.90 (TC_LWM_009)
                    ├── 0.95 (TC_LWM_010)
                    └── 1.0  (TC_LWM_012)

                    enable_coreference
                    ├── false (默认)
                    └── true  (TC_LWM_014, TC_LWM_016)

                    enable_llm_fallback
                    ├── false (默认)
                    └── true  (TC_LWM_015, TC_LWM_016)

                    linkable_types
                    ├── 不传/空 (全部类型)
                    ├── ["ORG"]     (TC_LWM_017)
                    ├── ["ORG","GPE"] (TC_LWM_018)
                    └── ["UNKNOWN"] (TC_LWM_020)

                    mentions 格式
                    ├── JSON 对象数组 (标准)
                    └── 逗号分隔字符串 (TC_LWM_006)

                    输入模式
                    ├── /link (原始文本→NER→链接)
                    └── /link_with_mentions (给定 mention→链接)
```
