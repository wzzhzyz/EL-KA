## 2. 消歧优先级规则

```
1. abbreviation 精确命中 + 单一候选 → 直接返回 (confidence=high)
2. alias 精确命中 + 多候选       → BGE语义排序
3. alias 模糊命中(ed<=1)         → BGE语义验证
4. 仅BGE语义召回                 → 低于nil_threshold→NIL
```

## 3. 已知badcase模式

| 模式 | 描述 | 建议 |
|------|------|------|
| 简称过短 | 2字简称多实体共享 | 扩充上下文窗口到100字 |
| 行业术语歧义 | "综合能源"既是术语也是业务描述 | 术语类型实体优先于业务描述匹配 |
| 数字混淆 | "3.8万亿"可能影响NER | NER边界不应包含数字 |
